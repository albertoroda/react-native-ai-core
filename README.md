# react-native-ai-core

High-performance AI bridge for React Native using Google AI Edge MediaPipe and Gemini Nano

## Installation


```sh
npm install react-native-ai-core
```


## Usage


```js
import { multiply } from 'react-native-ai-core';

// ...

const result = multiply(3, 7);
```
